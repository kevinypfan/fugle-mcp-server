# 最近 10 年股利資料 API

## API 用途
取得公司最近 10 年的股利發放記錄，包含現金股利、股票股利、除權息日期等完整股利資訊。此資料對於股利投資策略分析和公司配息政策研究非常重要。

## 主要功能
- 歷年現金股利金額（cshdiv）
- 股票股利資訊（stkdiv）
- 除權息相關日期
- 股利發放期間（季度）
- 每股盈餘（EPS）
- 配息比率（payoutRatio）
- 股利殖利率（divYield）
- 填息天數（fillDivDays）

## 使用場景
- 股利投資策略制定
- 公司配息政策分析
- 殖利率計算和比較
- 股利成長趨勢研究
- 投資組合配息規劃
